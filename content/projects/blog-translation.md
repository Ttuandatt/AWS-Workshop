## Overview

As part of my internship at AWS Vietnam, I translated **3 technical AWS blogs** from English to Vietnamese, making cloud knowledge more accessible to the Vietnamese developer community. Each translation required deep understanding of the technical concepts to ensure accuracy.

---

## Translated Blogs

### 1. Scalable Data Replication with Amazon FSx for NetApp ONTAP
A highly scalable solution for data replication using **Amazon FSx for NetApp ONTAP** and **NetApp SnapMirror**. Covers enterprise-grade storage solutions for hybrid and multi-cloud environments.

**Key Topics:**
- FSx for NetApp ONTAP architecture
- SnapMirror replication strategies
- Cross-region data protection
- Performance optimization for large-scale replication

---

### 2. Accelerating Aerospace Innovation with HPC on AWS
**High Performance Computing (HPC)** on Amazon Web Services for accelerating aerospace innovation. Covers compute-intensive workloads such as CFD simulations and structural analysis.

**Key Topics:**
- AWS HPC cluster architecture
- Auto Scaling for compute-intensive workloads
- Cost optimization for large-scale simulations
- Real-world aerospace industry use cases

---

### 3. Troubleshooting Elastic Beanstalk with Amazon Q Developer CLI
Using **Amazon Q Developer CLI** to diagnose and troubleshoot Elastic Beanstalk environment issues. A practical guide to AI-assisted cloud operations.

**Key Topics:**
- Amazon Q Developer CLI setup and usage
- Common Elastic Beanstalk troubleshooting scenarios
- AI-assisted debugging workflows
- Best practices for environment management

---

## Skills Gained

- **Technical Translation**: Translating complex cloud concepts while maintaining technical accuracy
- **AWS Knowledge**: Deep understanding of FSx, HPC, Elastic Beanstalk, and Amazon Q
- **Technical Writing**: Creating clear, accessible documentation in Vietnamese

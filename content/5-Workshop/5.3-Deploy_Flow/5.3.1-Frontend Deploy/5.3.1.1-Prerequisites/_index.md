### Cần chuẩn bị:

*   Tên miền đã đăng ký (VD: sgutodolist.com)
    - Đối với tài khoản Free Tier thì không được hỗ trợ tính năng đăng ký tên miền của dịch vụ Route 53, vì vậy project này đăng ký tên miền một nhà cung cấp tên miền khác và host tên miền ở Route
    
*   Tài khoản AWS
    
*   ReactJS project đã build thành công
    
*   AWS CLI (optional, để test)
    
*   Node.js & npm đã cài đặt
    

### Kiến thức cần có:

*   Hiểu cơ bản về S3, CloudFront, Route 53
    
*   Biết sử dụng AWS Console
    
*   Biết build ReactJS project

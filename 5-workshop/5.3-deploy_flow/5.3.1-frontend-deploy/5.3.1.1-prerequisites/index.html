Preparation Requirements

  • A registered domain name (e.g. sgutodolist.com)

    • With AWS Free Tier accounts, Route 53 domain registration is not supported.
      Therefore, in this project, the domain is registered with a third-party domain provider and then hosted on Amazon Route 53.

  • An AWS account

  • A successfully built ReactJS project

  • AWS CLI (optional, for testing and validation)

  • Node.js and npm installed on the local machine

Required Knowledge

  • Basic understanding of Amazon S3, CloudFront, and Route 53

  • Familiarity with using the AWS Management Console

  • Ability to build a ReactJS project
